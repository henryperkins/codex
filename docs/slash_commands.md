# Slash commands

For an overview of Codex CLI slash commands, see [this documentation](https://developers.openai.com/codex/cli/slash-commands).

Repository index controls also live in the TUI slash-command surface: `/index` shows `query_project`
defaults, `/index refresh [full]` runs `repo_index_refresh` for the current working directory, and
`/repo-index-refresh [full]` is the direct tool-name alias.
